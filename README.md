# 수학 DNA 연구소

수학 DNA 연구소는 수학축전 부스에서 참가자가 8개의 생활 속 수학 상황에 답하면 4글자 초성 DNA와 16개 유형 중 하나를 보여주는 모바일 우선 웹앱입니다.

## 실행 방법

웹앱 주소:

```text
https://deepseamath.github.io/math_DNA_lab/
```

로컬에서 확인하려면 `index.html` 파일을 브라우저로 열면 됩니다.

## 결과 저장 구조

검사 완료 결과는 Google Apps Script 웹앱을 통해 Google Sheets에 저장됩니다.

현재 앱의 저장 URL은 `index.html` 안의 `RESULTS_API_URL` 상수에 들어 있습니다.

저장되는 주요 값:

- `code`: 최종 수학 DNA 코드
- `typeName`: 16개 유형명
- `responses`: 8개 문항 응답 기록
- `createdAt`: 검사 완료 시각

## 문항 구조

총 8문항이며, 4개 영역별 2문항씩 구성됩니다.

앱 화면에서는 8문항의 순서가 매번 랜덤으로 섞입니다. 문항 순서는 달라지지만 각 문항이 속한 영역과 결과 산정 기준은 항상 동일합니다.

| 영역 | 문항 | 선택 A | 선택 B |
|---|---|---|---|
| 사고 시작 방식 | 편의점 1+1 상품 | 계산해 본다 `ㅅ` | 좋은 기회 같아 바로 고른다 `ㄱ` |
| 사고 시작 방식 | 친구들과 빙수 계산 | 인원수대로 계산한다 `ㅅ` | 비슷하게 나누면 된다고 생각한다 `ㄱ` |
| 문제 해결 방식 | 새 보드게임 | 설명서를 먼저 읽는다 `ㄴ` | 일단 시작하면서 익힌다 `ㅎ` |
| 문제 해결 방식 | 어려운 문제 | 조건을 하나씩 정리한다 `ㄴ` | 생각나는 방법부터 시도한다 `ㅎ` |
| 구조 인식 방식 | 벽돌 무늬 | 반복되는 규칙을 찾는다 `ㄹ` | 전체 모양과 배치를 본다 `ㄷ` |
| 구조 인식 방식 | 퍼즐 맞추기 | 규칙과 순서를 먼저 찾는다 `ㄹ` | 모양이 맞는지 먼저 확인한다 `ㄷ` |
| 정보 해석 방식 | 시험 결과 그래프 | 평균과 수치를 먼저 확인한다 `ㅂ` | 전체적인 추세를 먼저 본다 `ㅈ` |
| 정보 해석 방식 | 날씨 예보 자료 | 강수 확률 수치를 확인한다 `ㅂ` | 비가 올 것 같은 흐름을 본다 `ㅈ` |

## 영역과 성향

### 1. 사고 시작 방식

- `ㅅ` 수식형: 계산과 수치를 먼저 확인
- `ㄱ` 감각형: 직관과 느낌으로 먼저 판단

### 2. 문제 해결 방식

- `ㄴ` 논리형: 조건을 정리하고 순서대로 해결
- `ㅎ` 행동형: 일단 시도하며 해결

### 3. 구조 인식 방식

- `ㄹ` 규칙형: 반복과 패턴을 먼저 찾음
- `ㄷ` 도형형: 모양과 배치를 먼저 봄

### 4. 정보 해석 방식

- `ㅂ` 분석형: 표, 그래프, 수치를 확인
- `ㅈ` 직관형: 전체 흐름과 추세를 파악

## 결과 산정 방식

각 영역은 2문항으로 구성됩니다.

1. 각 영역별로 더 많이 선택된 성향을 해당 영역의 결과로 결정합니다.
2. 2문항이 1:1 동점이면 해당 영역의 마지막 응답을 기준으로 결정합니다.
3. 영역 순서는 항상 아래 순서로 고정됩니다.

```text
사고 시작 방식 + 문제 해결 방식 + 구조 인식 방식 + 정보 해석 방식
```

예시:

```text
ㅅ + ㄴ + ㄹ + ㅂ = ㅅㄴㄹㅂ
```
